Installation

Download

curl -O https://your-domain.com/public/sdk.js

Quick Start

// Initialize the client
const client = new DexAIClient({
  apiUrl: 'https://your-api-url.com'
});

// Connect wallet (Phantom/Solflare)
await client.connect();

// Make a paid API call
const result = await client.solanaAnalyzer.analyze('EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v');
console.log(result);

Wallet Connection

// Connect to Phantom or Solflare
const { connected, publicKey } = await client.connect();

// Check connection status
if (client.isConnected()) {
  console.log('Connected:', client.publicKey);
}

// Disconnect
await client.disconnect();

Error Handling

try {
  const result = await client.solanaAnalyzer.analyze(tokenMint);
} catch (error) {
  if (error instanceof PaymentRequiredError) {
    console.log('Payment needed:', error.payment.price, error.payment.currency);
  } else {
    console.error('Error:', error.message);
  }
}

x402 Protocol

All payments use the x402 micropayment protocol:

  1. Client makes API request
  2. Server returns 402 Payment Required with price
  3. Client signs payment with wallet
  4. Client resends request with X-Payment header
  5. Server verifies payment and returns response

The SDK handles this automatically when you're connected to a wallet.
